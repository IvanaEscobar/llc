NUMA Misses (foreign assigned): These are 4K pgs that overflow to another node when local memory is full.
NUMA Hits: Top(red) MBytes assigned from 4MB pgs; Bottom(blue) MBytes from 4K and 2MB pages.
NOTE: Hit Memory for 2MB pages (top) is shown over total memory (bottom) to show extent of 4MB usage.
              Best if they are equal-- all assigned memory is from 2MB pages
              (a ~5% variation may exist, even if equal, due to non instantaneous kernel updates).
NUMA Free: Total Free memory on all nodes (color areas show node portions).
NUMA Used: Total Used memory on all nodes (color areas show node portions).
